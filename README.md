# CodeLens

CodeLens is an AI-powered developer growth platform designed to help developers enhance their coding skills, streamline workflows, and prepare better for interviews. Built with a modern, scalable MERN architecture.

## Tech Stack
- **Frontend:** React + Vite, Tailwind CSS v4
- **Backend:** Node.js, Express, MongoDB
- **AI Integration:** Google Gemini API

## Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB connection string
- Gemini API Key

### Setup
1. Clone the repository.
2. Setup Environment Variables: Wait... See `frontend/.env` and `server/.env` to configure your keys.
3. Start Backend:
   ```bash
   cd server
   npm install
   npm start # Or node server.js
   ```
4. Start Frontend:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
